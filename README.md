# git-flow-test

## Feature branch

이 문장은 `git flow feature start edit-readme`라는 명령어로 새로운 `feature` 브랜치를 생성한 후 작성했습니다.

## Git-flow publish

이 문장을 수정한 후, `git flow feature publish for-publish`라는 명령어로 수정 내용을 원격 서버에 게시합니다.
